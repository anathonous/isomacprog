# isomacprog <br />
Convert 32 Bit Mac Pro Efi ISO's to Bios Only <br />
<br />
Use Prepackaged Bins or Compile. <br />
cc -g -Wall isomacprog.c.txt. -o isomacprog <br />
 <br />
cp original.iso macversion.iso <br />
./isomacprog macversion.iso <br />
